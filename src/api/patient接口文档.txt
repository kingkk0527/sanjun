患者分页查询
基本信息
Path： /patient/page
Method： GET

请求参数 Query

参数名称	    是否必须	 	备注
patientId	否	 	患者编号
name	    否	 	患者名称
page	    是	 	页码
pageSize	是	 	每页记录数

返回数据
名称	        类型	                    是否必须	默认值	备注	其他信息
code	    number	                必须
msg	        null	                非必须
data	    json	                非必须
├─ total	number	                必须		总记录数
├─ records	object []	            必须		当前页数据
    ├─ id	        integer	        必须
    ├─ name	        string	        必须
    ├─ patientId	string	        必须
    ├─ age  	    string	        必须
    ├─ sex  	    string	        必须
    ├─ image	    string	        必须
    ├─ updateTime   string	        必须
