
用户登录
Path： /employee/login
Method： POST

请求参数 body
名称	           类型	是否必须	默认值	备注	其他信息
password	string	必须		        密码
username	string	必须		        用户名

返回数据
名称	        类型	        是否必须	默认值	备注	其他信息
code	    integer	    必须
data	    json	    非必须		    用户登录返回的数据格式
├─ id	    integer	    非必须		    主键值
├─ name	    string	    非必须		    姓名
├─ userName	string	    非必须		    用户名
├─ phone	string	    非必须		    手机号
├─ password	string	    非必须		    密码
├─ role 	string	    必须		        用户权限   admin  管理员  user  普通用户
├─ token	string	    非必须		    登录安全控制token
msg	string	非必须

返回数据

用户分页查询
基本信息
Path： /employee/page
Method： GET

请求参数 Query

参数名称	    是否必须		    备注
name	    否	    	    用户姓名
page	    是	   	        页码
pageSize	是	    	    每页记录数

返回数据
名称	                类型	            是否必须	    默认值	备注	其他信息
code	            integer/number	必须
msg	                null	        非必须
data	            json	        必须
├─ total	        number	        必须
├─ records	        object []       必须            数据list
    ├─ id	        number	    必须            主键值
    ├─ username	string	        必须        账号
    ├─ name	    string	        必须        姓名
    ├─ password	string	        必须        密码
    ├─ phone	string	        必须        手机号
    ├─ role	    string	        必须        角色
